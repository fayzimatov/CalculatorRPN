# 📱 RPN Calculator

Этот проект – калькулятор, который при вычислении использует алгоритм **Shunting Yard**. 
Сначала он переводит выражение в **обратную польскую нотацию (RPN)**, а затем выполняет вычисления. 
Реализован с использованием архитектуры **MVVM**.

| Входные данные | Ожидаемый результат |
|---------------|------------------|
| `3 + 5 × 2`  | `13`            |
| `8 ÷ 0`      | `Ошибка`        |
| `0 ÷ 0`      | `Неопределенность` |

## 🔧 Функциональность

- **Удаление символов:**
  - Одно нажатие на кнопку "стереть" (**⌫**) удаляет **по одному символу**.
  - Долгое нажатие стирает **всё выражение**.

- **Поддерживаемые операции:** `+`, `-`, `×`, `÷`, `()`.

- **Обработка ошибок:**
  - Деление на `0` отображает `Ошибка`.
  - `0 ÷ 0` → `Неопределенность`.
